# PR Description Improver

Suggests improvements to pull request descriptions, focusing on clarity, completeness, and conciseness.

## Free
```yaml
- uses: walshd1/pr-description-improver@v1
  with:
    gemini_api_key: ${{ secrets.GEMINI_API_KEY }}
```

## Paid (cost + 4.75%)
```yaml
- uses: walshd1/pr-description-improver@v1
  with:
    service_token: ${{ secrets.ACTION_FACTORY_TOKEN }}
```
